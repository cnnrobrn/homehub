# Fun — Suggestions & Coordination

## Suggestions (catalog)

| Kind                | Trigger                                                                     |
|---------------------|-----------------------------------------------------------------------------|
| `outing_idea`       | Free weekend block + household preferences + seasonality                   |
| `trip_prep`         | Upcoming trip; draft packing list / tasks                                   |
| `book_reservation`  | Repeated pattern (e.g., family dinners out on Fridays); propose booking   |
| `revisit_venue`     | Loved venue (liked-flag) unvisited in N months                             |
| `try_new_venue`     | Repetition fatigue + memory-graph-informed candidates                       |

## Coordination

### Planning a joint outing

A Fun coordination card lets a member propose an outing with:

- Candidate attendees (household members + guests).
- Candidate dates (drawn from free windows across attendees' Google Calendars).
- Optional venue, with a "shortlist" generated by `outing_idea`.

Each attendee RSVPs in-app. Once confirmed, the outing becomes a real `app.event` and (optionally) mirrors to each attendee's Google Calendar.

### Trip planning

A trip's sub-events can be drafted together: flights, lodging, activities. HomeHub v1 does not book flights or hotels but collects the details and puts them on the calendar. Cost totals roll up into Financial when the member tags them.

## Execution

- `add_to_calendar` — member approval writes to HomeHub and optionally Google Calendar.
- `draft_reservation_request` — creates an email draft in the member's Gmail.
- `book_reservation` via provider API where supported (e.g., OpenTable partner API if added) — not in v1 unless a specific provider integration lands.
